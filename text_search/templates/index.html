OverScripted Explorer - Text Search

A demo tool that lets you search for text in certain columns of the OverScripted data set.

Results are displayed as a raw number of rows, which are individual javascript calls, and then grouped by the raw location and script_url and finally by the `netloc` from python's built-in urlparser (urlparse(url).netloc). Note that raw values may inlude queries, fragments, etc.

If you would like additional queries, options, views on the data, please open an issue.

Expect a 5% sample to take a few minutes.
